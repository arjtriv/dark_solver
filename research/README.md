# Research Workspace

This directory is the spillover zone for material that should stay out of `src/` until it hardens into production logic or public documentation.

Use it for:

- exploratory Z3 derivations
- solver heuristics under evaluation
- benchmark drafts and corpus notes
- pre-paper sketches, notebook exports, and experiment logs

Graduation rules:

- productionized engine logic moves into `src/`
- stable operator or architecture docs move into `docs/`
- reproducible findings move into `reports/`

Current subspaces:

- `z3/`: notes on symbolic encodings and Keccak abstractions
